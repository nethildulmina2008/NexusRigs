PC Build Guide (සිංහලෙන්)

1. කොටස් තෝරාගැනීම (Choosing Components)

පළමුව ඔබගේ අවශ්‍යතාවය (Gaming, Coding, හෝ Video Editing) සහ අයවැය (Budget) අනුව ගැලපෙන උපාංග තෝරාගත යුතුය. ප්‍රධාන වශයෙන් අවශ්‍ය වන කොටස්:

  • ප්‍රොසෙසරය (Processor/CPU): Intel හෝ AMD.
  • මවු පුවරුව (Motherboard): CPU එකට ගැලපෙන Socket එකක් සහිත මවු පුවරුවක් විය යුතුය.
  • මතකය (RAM): අවම 8GB හෝ 16GB (DDR4/DDR5).
  • ගබඩාව (Storage): වේගවත් ක්‍රියාකාරීත්වයට NVMe SSD එකක් නිර්දේශ කෙරේ.
  • ග්‍රැෆික් කාඩ්පත (Graphics Card/GPU): Gaming හෝ Graphic වැඩ සඳහා.
  • බල සැපයුම (PSU): මුළු පද්ධතියටම ප්‍රමාණවත් Watt ගණනක් සහිත එකක්.
  • කවරය (Casing): සියලු කොටස් ඇතුළත් කළ හැකි ප්‍රමාණයේ එකක්.

2. මවු පුවරුව සූදානම් කිරීම (Motherboard Setup)

කේසිං එකට දැමීමට පෙර මවු පුවරුව මත කොටස් සවි කිරීම පහසුය.

  • CPU එක සවි කිරීම: මවු පුවරුවේ CPU socket එක විවෘත කර, CPU එකේ ඇති සලකුණු වලට අනුව ඉතා ප්‍රවේශමෙන් තබන්න.
  • RAM සවි කිරීම: RAM slots වල ලොක් එක විවෘත කර "ක්ලික්" හඬ ඇසෙන තෙක් තද කරන්න.
  • SSD සවි කිරීම: M.2 SSD එකක් නම් අදාළ slot එකට ඇතුළු කර කුඩා ඇණයක් මඟින් සවි කරන්න.

3. CPU Cooler එක සවි කිරීම

CPU එක රත් වීම පාලනය කිරීමට මෙය අත්‍යවශ්‍යයි.

  • CPU එක මත Thermal Paste ස්වල්පයක් තවරා ඒ මත Cooler එක තබා තද කරන්න. (සමහර Cooler වල thermal paste කලින්ම තවරා ඇත).
  • Cooler එකේ වයරය "CPU_FAN" ලෙස සඳහන් පින් එකට සම්බන්ධ කරන්න.

4. කේසිං එක තුළට මවු පුවරුව සවි කිරීම

  • කේසිං එකේ පිටුපස ඇති සිදුරට I/O Shield එක සවි කරන්න.
  • මවු පුවරුව කේසිං එක ඇතුළත තබා ඇණ (Screws) යොදා නිසි පරිදි සවි කරන්න.

5. බල සැපයුම (PSU) සහ GPU සවි කිරීම

  • PSU: කේසිං එකේ අදාළ ස්ථානයේ PSU එක තබා සවි කරන්න.
  • GPU: මවු පුවරුවේ ඇති PCI-E slot එකට ග්‍රැෆික් කාඩ්පත සවි කර කේසිං එකට ඇණ මඟින් සම්බන්ධ කරන්න.

6. වයර් සම්බන්ධ කිරීම (Cabling)

මෙය වඩාත් සැලකිලිමත් විය යුතු පියවරයි:

  • මවු පුවරුවට (24-pin) සහ CPU එකට (4/8-pin) බලය ලබා දෙන වයර් සම්බන්ධ කරන්න.
  • GPU එකට අවශ්‍ය බල සැපයුම් වයර් සම්බන්ධ කරන්න.
  • කේසිං එකේ ඉදිරිපස ඇති Buttons (Power, Reset) සහ USB ports වල වයර් මවු පුවරුවේ අදාළ පින් වලට සම්බන්ධ කරන්න.

7. පරීක්ෂා කිරීම (Testing)

  • පරිගණකය බල සැපයුමට සම්බන්ධ කර Power Button එක ක්‍රියාත්මක කරන්න.
  • Fan කැරකෙනවා නම් සහ Monitor එකේ Display එකක් දිස් වේ නම් ඔබ සාර්ථකව PC එක එකලස් කර ඇත.
  • ඉන්පසු Windows OS එක ඇතුළත් කර Drivers Update කරගන්න.
විශේෂ උපදේශය: ස්ථිතික විද්‍යුතයෙන් (Static Electricity) කොටස් වලට හානි විය හැකි බැවින්, වැඩ ආරම්භ කිරීමට පෙර ලෝහමය මතුපිටක් ස්පර්ශ කර සිරුරේ ආරෝපණ ඉවත් කරගන්න.
